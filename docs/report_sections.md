# 三维建模实习报告素材

## 4.3 结果

本实验以 `20251126150027848.las` 作为处理对象，围绕激光点云地形三维建模完成了数据读取、点云去噪、地面点滤波、地面点成果输出和 DEM 构建。原始点云共包含 `9,252,140` 个点，经过高程分位数约束去噪后，参与滤波处理的有效点数为 `9,233,733` 个。去噪过程剔除了明显偏离主体高程范围的异常点，降低了离群点对地面滤波和 DEM 插值的影响。

在地面点提取阶段，实验分别采用渐进式三角网滤波思想和布料滤波思想进行处理。渐进式三角网滤波结果提取地面点 `953,376` 个，该结果对高于地表的非地面点剔除较明显，适合作为后续 DEM 构建的数据基础。布料滤波结果提取地面点 `8,233,990` 个，保留的点数更多，地表连续性较好，但相较于渐进式三角网滤波结果，局部区域可能保留更多接近地表的非地面点。两种滤波结果均已保存为地面点 LAS 成果，分别为 `ground_tin.las` 和 `ground_csf.las`，可用于后续对比分析。

在 DEM 构建阶段，实验选用渐进式三角网滤波得到的地面点作为主要输入，按 `0.5 m` 分辨率进行规则格网化插值，生成数字高程模型。最终 DEM 栅格尺寸为 `267 x 174`，成果保存为 `dem_tin.tif`，并同步输出 `dem_tin.tfw` 地理配准文件。从 DEM 预览结果看，地形表面整体连续，能够反映堤防区域及周边地形的高程变化特征。

本次实验最终形成了原始点云预览图、地面滤波对比图、DEM 预览图、地面点 LAS 文件和 DEM GeoTIFF 文件等成果。为便于成果复现与后续学习，本实验将处理代码、原始 LAS 数据、非 PPTX 结果文件、算法流程图和说明文档整理为开源项目。其中，体量较大的 LAS 文件通过 Git LFS 进行管理；PPTX 文件作为本地报告汇总材料，不纳入开源成果。复现实验时，可直接运行仓库中的处理脚本，也可基于 `data/` 和 `outputs/` 中的数据与成果进行检查和对比。
